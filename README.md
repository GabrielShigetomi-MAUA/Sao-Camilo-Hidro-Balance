# S-o-Camilo-Hidro-Balance
Aplicativo multiplataforma para avaliação padronizada da taxa de sudorese e suporte à tomada de decisão em hidratação de atletas. Permite coletar, calcular e monitorar o balanço hídrico em sessões de treino e competição, gerando recomendações individualizadas de ingestão de fluidos e eletrólitos.
